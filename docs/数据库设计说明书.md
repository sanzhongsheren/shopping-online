## 1 数据库概述

### 1.1 项目背景

本数据库服务于商品网上交易系统，采用MySQL关系型数据库存储系统核心业务数据。

### 1.2 数据库信息


| 属性       | 值                 |
| ---------- | ------------------ |
| 数据库名称 | online_shopping    |
| 数据库类型 | MySQL 8.0+         |
| 字符集     | utf8mb4            |
| 排序规则   | utf8mb4_unicode_ci |
| 存储引擎   | InnoDB             |

### 1.3 核心业务模块

根据需求分析，系统包含5大核心模块：

1. **用户管理**：用户注册、登录、角色区分（买家/卖家/管理员）
2. **商品管理**：商品发布、分类、库存、图片管理
3. **审查管理**：用户资质审核、商品合规性审核（双审核机制）
4. **用户购物**：购物车管理
5. **订单配送管理**：订单全流程状态流转（待支付→已支付→已发货→已收货→已完成）

### 2.2 核心实体说明


| 实体        | 说明                         | 主要关系                                                                 |
| ----------- | ---------------------------- | ------------------------------------------------------------------------ |
| users       | 系统用户（买家/卖家/管理员） | 1:N→addresses, 1:N→products, 1:N→orders, 1:N→cart                    |
| addresses   | 用户收货地址                 | N:1→users, 1:N→orders                                                  |
| categories  | 商品分类（支持二级分类）     | 1:N→products, 自关联                                                    |
| products    | 商品信息                     | N:1→users, N:1→categories, 1:N→order_items, 1:N→cart, 1:N→favorites |
| orders      | 订单主表                     | N:1→users, N:1→addresses, 1:N→order_items                             |
| order_items | 订单明细（商品快照）         | N:1→orders, N:1→products                                               |
| audits      | 审核记录表                   | N:1→users(applicant), N:1→users(auditor)                               |
| messages    | 用户留言                     | N:1→users, 自关联(回复)                                                 |
| cart        | 购物车                       | 复合主键(user_id, product_id)                                            |
| favorites   | 收藏夹                       | 复合主键(user_id, product_id)                                            |

## 3 数据表详细设计

### 3.1 用户表 (users)


| 字段名        | 数据类型                                                  | 约束                        | 说明                   |
| ------------- | --------------------------------------------------------- | --------------------------- | ---------------------- |
| user_id       | INT                                                       | PK, AUTO_INCREMENT          | 用户ID                 |
| username      | VARCHAR(50)                                               | NOT NULL, UNIQUE            | 用户名                 |
| password      | VARCHAR(255)                                              | NOT NULL                    | 密码（bcrypt加密存储） |
| email         | VARCHAR(100)                                              | -                           | 邮箱                   |
| phone         | VARCHAR(20)                                               | -                           | 手机号                 |
| role          | ENUM('buyer','seller','admin')                            | NOT NULL, DEFAULT 'buyer'   | 角色                   |
| status        | ENUM('pending','approved','rejected','active','disabled') | NOT NULL, DEFAULT 'pending' | 账号状态               |
| real_name     | VARCHAR(50)                                               | -                           | 真实姓名               |
| id_card       | VARCHAR(18)                                               | -                           | 身份证号（卖家审核用） |
| avatar_url    | VARCHAR(255)                                              | -                           | 头像URL                |
| created_at    | TIMESTAMP                                                 | DEFAULT CURRENT_TIMESTAMP   | 创建时间               |
| updated_at    | TIMESTAMP                                                 | ON UPDATE CURRENT_TIMESTAMP | 更新时间               |
| last_login_at | TIMESTAMP                                                 | NULL                        | 最后登录时间           |

**索引**：`idx_username`, `idx_role`, `idx_status`

**业务规则**：

- 密码使用bcrypt加密存储
- 卖家注册需提交身份证信息待审核
- 新用户默认状态为pending，需管理员审核

---

### 3.2 收货地址表 (addresses)


| 字段名         | 数据类型     | 约束                        | 说明         |
| -------------- | ------------ | --------------------------- | ------------ |
| address_id     | INT          | PK, AUTO_INCREMENT          | 地址ID       |
| user_id        | INT          | FK→users, NOT NULL         | 用户ID       |
| receiver_name  | VARCHAR(50)  | NOT NULL                    | 收货人姓名   |
| phone          | VARCHAR(20)  | NOT NULL                    | 联系电话     |
| province       | VARCHAR(50)  | NOT NULL                    | 省份         |
| city           | VARCHAR(50)  | NOT NULL                    | 城市         |
| district       | VARCHAR(50)  | NOT NULL                    | 区县         |
| detail_address | VARCHAR(255) | NOT NULL                    | 详细地址     |
| postal_code    | VARCHAR(10)  | -                           | 邮政编码     |
| is_default     | BOOLEAN      | DEFAULT FALSE               | 是否默认地址 |
| created_at     | TIMESTAMP    | DEFAULT CURRENT_TIMESTAMP   | 创建时间     |
| updated_at     | TIMESTAMP    | ON UPDATE CURRENT_TIMESTAMP | 更新时间     |

**索引**：`idx_user_id`

**业务规则**：

- 用户可添加多个收货地址
- 订单关联地址时使用RESTRICT策略，防止删除已用地址

---

### 3.3 商品分类表 (categories)


| 字段名        | 数据类型    | 约束                      | 说明                   |
| ------------- | ----------- | ------------------------- | ---------------------- |
| category_id   | INT         | PK, AUTO_INCREMENT        | 分类ID                 |
| category_name | VARCHAR(50) | NOT NULL                  | 分类名称               |
| parent_id     | INT         | FK→categories, NULL      | 父分类ID（顶级为NULL） |
| sort_order    | INT         | DEFAULT 0                 | 排序顺序               |
| created_at    | TIMESTAMP   | DEFAULT CURRENT_TIMESTAMP | 创建时间               |

**索引**：`idx_parent_id`

**业务规则**：

- 支持二级分类结构
- parent_id为NULL表示顶级分类
- sort_order用于控制同级分类显示顺序

---

### 3.4 商品表 (products)


| 字段名       | 数据类型                                                           | 约束                        | 说明         |
| ------------ | ------------------------------------------------------------------ | --------------------------- | ------------ |
| product_id   | INT                                                                | PK, AUTO_INCREMENT          | 商品ID       |
| seller_id    | INT                                                                | FK→users, NOT NULL         | 卖家ID       |
| category_id  | INT                                                                | FK→categories, NULL        | 分类ID       |
| product_name | VARCHAR(100)                                                       | NOT NULL                    | 商品名称     |
| description  | TEXT                                                               | -                           | 商品描述     |
| price        | DECIMAL(10,2)                                                      | NOT NULL                    | 单价         |
| stock        | INT                                                                | NOT NULL, DEFAULT 0         | 库存数量     |
| unit         | VARCHAR(20)                                                        | DEFAULT '件'                | 计量单位     |
| image_url    | VARCHAR(255)                                                       | -                           | 商品主图URL  |
| images       | JSON                                                               | -                           | 商品图片列表 |
| status       | ENUM('pending','approved','rejected','onsale','offsale','deleted') | NOT NULL, DEFAULT 'pending' | 商品状态     |
| view_count   | INT                                                                | DEFAULT 0                   | 浏览次数     |
| sales_count  | INT                                                                | DEFAULT 0                   | 销量         |
| created_at   | TIMESTAMP                                                          | DEFAULT CURRENT_TIMESTAMP   | 创建时间     |
| updated_at   | TIMESTAMP                                                          | ON UPDATE CURRENT_TIMESTAMP | 更新时间     |

**索引**：`idx_seller_id`, `idx_category_id`, `idx_status`, `idx_product_name`

**业务规则**：

- 商品发布后状态为pending，需管理员审核
- status='onsale'表示上架销售中
- price使用DECIMAL(10,2)保证精度
- images字段存储JSON数组，支持多图展示

---

### 3.5 购物车表 (cart)


| 字段名     | 数据类型  | 约束                        | 说明           |
| ---------- | --------- | --------------------------- | -------------- |
| cart_id    | INT       | PK, AUTO_INCREMENT          | 购物车ID       |
| user_id    | INT       | FK→users, NOT NULL         | 用户ID（买家） |
| product_id | INT       | FK→products, NOT NULL      | 商品ID         |
| quantity   | INT       | NOT NULL, DEFAULT 1         | 购买数量       |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP   | 添加时间       |
| updated_at | TIMESTAMP | ON UPDATE CURRENT_TIMESTAMP | 更新时间       |

**唯一约束**：`uk_user_product(user_id, product_id)`

**索引**：`idx_user_id`

**业务规则**：

- 同一用户对同一商品只能有一条购物车记录
- quantity表示该商品购买数量

---

### 3.6 订单主表 (orders)


| 字段名         | 数据类型                                                                               | 约束                                | 说明       |
| -------------- | -------------------------------------------------------------------------------------- | ----------------------------------- | ---------- |
| order_id       | INT                                                                                    | PK, AUTO_INCREMENT                  | 订单ID     |
| order_no       | VARCHAR(32)                                                                            | NOT NULL, UNIQUE                    | 订单号     |
| buyer_id       | INT                                                                                    | FK→users, NOT NULL                 | 买家ID     |
| address_id     | INT                                                                                    | FK→addresses, NOT NULL             | 收货地址ID |
| total_amount   | DECIMAL(10,2)                                                                          | NOT NULL                            | 订单总金额 |
| status         | ENUM('pending_payment','paid','shipped','received','completed','cancelled','refunded') | NOT NULL, DEFAULT 'pending_payment' | 订单状态   |
| payment_method | VARCHAR(20)                                                                            | -                                   | 支付方式   |
| payment_time   | TIMESTAMP                                                                              | NULL                                | 支付时间   |
| shipping_time  | TIMESTAMP                                                                              | NULL                                | 发货时间   |
| received_time  | TIMESTAMP                                                                              | NULL                                | 收货时间   |
| cancel_time    | TIMESTAMP                                                                              | NULL                                | 取消时间   |
| cancel_reason  | VARCHAR(255)                                                                           | -                                   | 取消原因   |
| buyer_message  | VARCHAR(255)                                                                           | -                                   | 买家留言   |
| created_at     | TIMESTAMP                                                                              | DEFAULT CURRENT_TIMESTAMP           | 创建时间   |
| updated_at     | TIMESTAMP                                                                              | ON UPDATE CURRENT_TIMESTAMP         | 更新时间   |

**索引**：`idx_order_no`, `idx_buyer_id`, `idx_status`, `idx_created_at`

**订单状态流转**：

```
pending_payment → paid → shipped → received → completed
      ↓
  cancelled

paid → refunded（退款）
```

**业务规则**：

- order_no格式：ORD+年月日+6位序号（如ORD202605010001）
- 地址使用RESTRICT策略，订单关联的地址不可删除

---

### 3.7 订单明细表 (order_items)


| 字段名        | 数据类型      | 约束                      | 说明             |
| ------------- | ------------- | ------------------------- | ---------------- |
| item_id       | INT           | PK, AUTO_INCREMENT        | 明细ID           |
| order_id      | INT           | FK→orders, NOT NULL      | 订单ID           |
| product_id    | INT           | FK→products, NOT NULL    | 商品ID           |
| product_name  | VARCHAR(100)  | NOT NULL                  | 商品名称（快照） |
| product_image | VARCHAR(255)  | -                         | 商品图片（快照） |
| price         | DECIMAL(10,2) | NOT NULL                  | 单价（快照）     |
| quantity      | INT           | NOT NULL                  | 购买数量         |
| subtotal      | DECIMAL(10,2) | NOT NULL                  | 小计金额         |
| created_at    | TIMESTAMP     | DEFAULT CURRENT_TIMESTAMP | 创建时间         |

**索引**：`idx_order_id`, `idx_product_id`

**业务规则**：

- 存储商品快照信息，下单时的价格和名称
- 后续商品价格变动不影响已生成订单

---

### 3.8 审核记录表 (audits)


| 字段名        | 数据类型                                               | 约束                        | 说明     |
| ------------- | ------------------------------------------------------ | --------------------------- | -------- |
| audit_id      | INT                                                    | PK, AUTO_INCREMENT          | 审核ID   |
| audit_type    | ENUM('user_register','product_publish','product_edit') | NOT NULL                    | 审核类型 |
| target_type   | ENUM('user','product')                                 | NOT NULL                    | 目标类型 |
| target_id     | INT                                                    | NOT NULL                    | 目标ID   |
| applicant_id  | INT                                                    | FK→users, NOT NULL         | 申请人ID |
| auditor_id    | INT                                                    | FK→users, NULL             | 审核人ID |
| status        | ENUM('pending','approved','rejected')                  | NOT NULL, DEFAULT 'pending' | 审核状态 |
| reject_reason | VARCHAR(255)                                           | -                           | 驳回原因 |
| audit_comment | VARCHAR(255)                                           | -                           | 审核备注 |
| submitted_at  | TIMESTAMP                                              | DEFAULT CURRENT_TIMESTAMP   | 提交时间 |
| audited_at    | TIMESTAMP                                              | NULL                        | 审核时间 |

**索引**：`idx_target(target_type,target_id)`, `idx_status`, `idx_applicant_id`

**业务规则**：

- 卖家注册需审核用户资质（audit_type='user_register'）
- 商品发布需审核合规性（audit_type='product_publish'）
- 商品编辑后需重新审核（audit_type='product_edit'）

### 3.9 留言表 (messages)


| 字段名      | 数据类型                                | 约束                        | 说明               |
| ----------- | --------------------------------------- | --------------------------- | ------------------ |
| message_id  | INT                                     | PK, AUTO_INCREMENT          | 留言ID             |
| user_id     | INT                                     | FK→users, NOT NULL         | 留言用户ID         |
| target_type | ENUM('user','product','order','system') | NOT NULL                    | 留言对象类型       |
| target_id   | INT                                     | -                           | 留言对象ID         |
| content     | TEXT                                    | NOT NULL                    | 留言内容           |
| parent_id   | INT                                     | FK→messages, NULL          | 父留言ID（回复用） |
| status      | ENUM('normal','hidden','deleted')       | DEFAULT 'normal'            | 状态               |
| created_at  | TIMESTAMP                               | DEFAULT CURRENT_TIMESTAMP   | 留言时间           |
| updated_at  | TIMESTAMP                               | ON UPDATE CURRENT_TIMESTAMP | 更新时间           |

**索引**：`idx_target(target_type,target_id)`, `idx_user_id`, `idx_parent_id`

**业务规则**：

- 支持对商品、订单进行留言
- parent_id用于支持留言回复（树形结构）
- system类型的target_id可为NULL

## 5 附录

### 5.1 约束说明


| 约束类型       | 说明             |
| -------------- | ---------------- |
| PK             | Primary Key 主键 |
| FK             | Foreign Key 外键 |
| NOT NULL       | 非空约束         |
| UNIQUE         | 唯一约束         |
| DEFAULT        | 默认值           |
| AUTO_INCREMENT | 自增主键         |
| ENUM           | 枚举类型         |

### 5.2 级联操作策略


| 父表       | 子表           | 策略                                  |
| ---------- | -------------- | ------------------------------------- |
| users      | addresses      | CASCADE                               |
| users      | products       | CASCADE                               |
| users      | orders         | CASCADE                               |
| users      | cart           | CASCADE                               |
| users      | favorites      | CASCADE                               |
| users      | messages       | CASCADE                               |
| users      | audits         | CASCADE(applicant), SET NULL(auditor) |
| categories | products       | SET NULL                              |
| products   | cart           | CASCADE                               |
| products   | order_items    | RESTRICT                              |
| products   | favorites      | CASCADE                               |
| orders     | order_items    | CASCADE                               |
| addresses  | orders         | RESTRICT                              |
| messages   | messages(回复) | CASCADE                               |

### 5.3 命名规范

- 表名：英文小写，单数形式，如 `users` 而非 `user_list`
- 字段名：英文小写，下划线分隔，如 `user_id`, `created_at`
- 索引名：`idx_` + 字段名，如 `idx_user_id`
- 外键名：`fk_` + 父表名 + `_` + 子表名
